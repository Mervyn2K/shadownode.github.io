Joining ShadowNode

If you're looking to join ShadowNode and unsure how simply follow the steps below. If you follow these steps and still have issues, we typically can offer some advice in our Discord by using our Support section.

  1. Find your desired pack image and double click to start loading the pack.
  2. Wait a few moments for the pack to start loading. You will see something like this show up after a short while. This process can take several minutes.
  3. Once the pack is loaded click the "Multiplayer" button. This might be labeled different if you're not using English as your computer's language.
  4. Accept the multiplayer warning by clicking the checkbox and then "Proceed".
  5. Allow Java and Minecraft through the Firewall.
  6. Click Add Server if you don't have our server already. You can get the desired pack-specific server IP here. Enter the server IP in the box next to the 1. Click Done when complete.
  7. Click the ShadowNode server you just added in your list. It should look something like the top box. After that click the Join button highlighted towards the bottom of your screen.
  8. Wait while the server and your client talk and you are joined to the server. If anything goes wrong at this point please visit our Discord and use our support section for further help .
